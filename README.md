# Biometric Attendance System

A comprehensive biometric attendance system with a Django backend and a React (Vite) frontend.

## Project Structure

- **[backend/](./backend)**: Django REST Framework API, face recognition services, and database management.
- **[frontend/](./frontend)**: React application with Tailwind CSS for student and host dashboards.

## Getting Started

Refer to the README files in the respective directories for setup and development instructions.

- [Backend Setup](./backend/README.md)
- [Frontend Setup](./frontend/README.md)

## Deployment

The project is configured for deployment on platforms like Render or Heroku.
- The `backend/` directory contains a `build.sh` script and `Procfile`.
- The `frontend/` directory is a static site ready for building via `npm run build`.
